# تبدیل تاریخ میلادی به شمسی به صورت Live و استفاده از Picker شمسی به جای میلادی!

این افزونه تاریخ ها و ساعت های نمایش داده شده به کاربر در سامانه Wallix را خودکار و لایو به شمسی تبدیل میکند و به همراه روز (مثلا شنبه 19 اردیبهشت 1405 - 09:30) نمایش میدهد. 
با این ابزار میتوانید به راحتی تاریخ ها نمایش داده را به شمسی مشاهده نمایید. به طوری کاربر تصور میکنه تاریخ ها از ابتدا فارسی نمایش داده میشوند و تبدیل شده به شمسی!


همچنین میتوانید از یک Picker یا انتخاب گر ترایخ شمسی نیز برای ورود تاریخ انقضای کاربر استفاده نمایید. شما تاریخ رو به شمسی انتخاب کرده و افزونه به طور خودکار به میلادی تبدیل و ثبت مینماید.
برای عدم تداخل با سایر سامانه ها و Web page ها، میبایست لینک سامانه رو وارد افزونه کنید، اینجوری این افزونه فقط روی صفحات اون هاست اعمال میشه


همچنین میتونید 3 تا پروفایل اضافه کنید (اختیاری) که از این تاریخ ها به عنوان دسترسی سریع (Quick Access) موقع ورود تاریخ توی Picker استفاده کنید و لازم نباشه مرتبا دستی تاریخ رو انتخاب کنید!
